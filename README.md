# Quiz-Project

A modern, interactive quiz game built using HTML, CSS, JavaScript, Bootstrap, and the **Open Trivia Database API.
This project features category-based quizzes, real-time questions, score tracking, sound effects, and celebratory confetti for a perfect score.
________________________________________________________________________________________________________________________________________________________
## Features

- Multiple quiz categories:
  - General Knowledge
  - Programming
  - Science
  - Sports
  - History
  - Geography
